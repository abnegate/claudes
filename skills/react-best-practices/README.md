# react-best-practices

React development guidelines with hooks, component patterns, state management, and performance optimization.

## What it covers

A reference skill that surfaces modern React guidance when building applications, reviewing React code, or optimising performance. Patterns are drawn from [Vercel's agent-skills](https://github.com/vercel-labs/agent-skills).

## Topics

- Hooks usage and rules
- Component patterns and composition
- State management
- Performance optimisation
- TypeScript and Next.js conventions

## When to use

- Building React applications
- Learning React best practices
- Code reviews for React projects
- Performance optimisation
