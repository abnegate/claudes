# frontend-design

Create distinctive, production-grade frontend interfaces with high design quality that avoid generic AI aesthetics.

Reference material loaded by Claude when building web components, pages, artifacts, posters, or applications (landing pages, dashboards, React components, HTML/CSS layouts, or any UI styling work). There is no slash command — the skill shapes how Claude approaches frontend work in context.

## What it covers

- Design thinking: committing to a bold aesthetic direction (brutalist, editorial, retro-futuristic, luxury, playful, etc.) grounded in purpose, tone, constraints, and a memorable differentiator.
- Typography guidance: picking distinctive fonts and avoiding generic defaults like Arial, Inter, and Roboto.
- Color and theme: cohesive palettes driven by CSS variables, with dominant colors and sharp accents instead of timid even distributions.
- Motion: high-impact moments and orchestrated reveals over scattered micro-interactions.
- Spatial composition: asymmetry, overlap, diagonal flow, grid-breaking, and intentional use of negative space or density.
- Backgrounds and visual detail: atmosphere and depth instead of flat solid fills.
- Anti-patterns: cliched purple-on-white gradients, predictable layouts, and other AI-generated tropes to avoid.
